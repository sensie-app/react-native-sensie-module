# react-native-sensie-module

Sensie SDK

## Installation

```sh
npm install react-native-sensie-module
```

## Usage

```js
import { whipCounter } from "react-native-sensie-module";

// ...

const result = await whipCounter({yaw: [1.0, 2.0, 3.0]})
```
